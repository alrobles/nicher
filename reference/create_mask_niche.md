# Crear una máscara para parámetros de modelo de nicho gaussiano

Crear una máscara para parámetros de modelo de nicho gaussiano

## Usage

``` r
create_mask_niche(p)
```

## Arguments

- p:

  Número de variables ambientales.

## Value

Vector nombrado con NAs para los parámetros mu y los elementos de la
triangular inferior de L.
