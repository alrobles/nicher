# get_env_var Function to extract environmental values from presence points

get_env_var Function to extract environmental values from presence
points

## Usage

``` r
get_env_var(df, env)
```

## Arguments

- df:

  A data frame with presence points

- env:

  A raster with environmental layers. Must have a projection string

## Examples

``` r
stack_path <- system.file("extdata", "stack_1_12_crop.rds", package="nicher")
stack_1_12 <- terra::unwrap(readr::read_rds(stack_path))
get_env_var(rawSpOccPnts, stack_1_12)
```
