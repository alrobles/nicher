# Predict method for "ellipse" class

Predict method for "ellipse" class

## Usage

``` r
# S3 method for class 'ellipse'
predict(object, layers, returnTable = FALSE, ...)
```

## Arguments

- object:

  An object of class "ellipse"

- layers:

  An object of class "rast" from terra package where to evaluate the
  ellipse model

- returnTable:

  logical if true return table instead a raster with the prediction

- ...:

  Other arguments passed to or from other methods
