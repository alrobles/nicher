Samples points from M hypothesis to estimate negative log likelihood
from Abeillia abeillei presence points. This is a hummingbird example. A
dataset with two variables containing points contains 2 bioclimatic
variables \#' @format A data frame with 10000 rows and 2 variables:

- bio1WH:

  temperature, in C

- bio12WH:

  precipitaion, in mm

## Usage

``` r
samMPts
```

## Format

An object of class `matrix` (inherits from `array`) with 10000 rows and
2 columns.
